Etiquetas

La etiqueta "br" cambia la linea.
El elemento "p" representa un párrafo.
La etiqueta "b/strong" pone en negrilla un texto.
La etiqueta "u/int" texto subrayado.
Eiqueta "i/em" texto marcado con un estilo en cursiva o italica.
Hace el tamaño del texto una talla más pequeña.
Etiqueta "big" tamaño del texto mas grande.
Etiqueta "sub"representa un subíndice
Etiqueta "sup"representa superindice.
Etiqueta "strike" Muestra el texto tachado.

Etiqueta "pre" utilizado
            para mostrar código 
            preformateado del editor,
                        texto plano o arte ascii

Etiqueta "abbr."nos permite añadir abreaviaturas a palabras del texto.
Elemento "mark"representa un texto marcado o resaltado.
Etiqueta "a" "Tutorial enlances"nos ayuda a poder crear un enlace a una página web.
Etiqueta "img" representa una imagen en el documento.
Etiqueta "ul" permite definir una lista desordenada y "ol" lista ordenada.
Etiqueta "blockquote" crea citas en bloque, marca las citas a otros autores o documentos.
Etiqueta "h1" crea un titulo.
